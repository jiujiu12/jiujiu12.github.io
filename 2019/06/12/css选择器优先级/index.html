发表于 2019-06-12 更新于 2020-08-21 阅读次数：

CSS样式优先级是如何排列的，以及权重是多少呢？

优先级排列及权重

​ !important > 行内样式 > id选择器* （100）> class选择器 | 属性选择器 | 伪类选择器（10）> 标签选择器（HTML标签，伪元素选择器）（1）

特殊情况

  1. ：not（）时，根据里面的选择器权重判断权重

  2. 一般不能跨级比较权重，特殊情况：有时候256个class才能干掉一个id

  3. 最小宽度，最大宽度，最小高度，最大高度这几个都比important高
